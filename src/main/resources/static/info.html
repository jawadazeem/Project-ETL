About This Billing Dashboard

Overview

This dashboard is a demonstration project for exploring how telecom billing data can be displayed, summarized, and analyzed in a clean, modern interface. It shows how records can be loaded, filtered, and ranked, and how overall charge patterns can be highlighted for insight.

What You Can Do Here
  • Load a real-time summary of all billing data
  • Browse individual telecom records in a sortable table
  • Filter records by state for targeted review
  • View the highest-charging accounts with Top-N queries
  • Explore charge patterns and data usage across all senators
Technologies Used
  • Java 17 – Core language used to implement the ETL pipeline, service layer, model layer, and backend logic.
  • Spring Boot – Framework for building the REST API, handling dependency injection, managing application lifecycle, and structuring the project using standard enterprise patterns.
  • Spring Web (Spring MVC) – Exposes REST endpoints, handles HTTP routing, request/response serialization, and controller logic.
  • SLF4J + Logback – Logging for observability, error tracking, and debugging across ETL, service, and controller layers.
  • JUnit 5 – Unit testing framework for validating ETL logic, service behavior, and controller endpoints.
  • Mockito – Mocking framework used to simulate service dependencies in controller tests and isolate components under test.
  • MockMvc – Spring MVC test framework used to send simulated HTTP requests and validate controller responses without running a full server.
  • OpenCSV – Library for parsing CSV input files as part of the ETL (Extract, Transform, Load) process.
  • AWS Elastic Beanstalk – Platform used to deploy and host the live Spring Boot application in a scalable and managed environment.
  • AWS S3 – Used for CSV uploads via pre-signed URLs, allowing users to securely upload billing files directly to cloud storage.
  • Maven – Build and dependency management tool used to manage libraries, run tests, and package the application.
  • HTML5 + Bootstrap – Frontend UI for viewing analytics, summaries, and interactive filtering of billing records.
